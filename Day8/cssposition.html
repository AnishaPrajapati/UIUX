CSS Position

  1. Static => default position, can't use top, bottom, left, right properties
  2. Relative => relative to it's normal position, can not affect other ekement
  3. Absolute => relative to nearest parent position
  4. Fixed => relative to the viewport
  5. Sticky => when you scroll the page
